# 📊 Financial Market Analysis using News Sentiment & Sector Classification

## 🚀 Project Overview

This project analyzes financial news articles to:

* Classify them into market sectors (IT, Banking, Pharma, etc.)
* Detect sentiment (Positive, Negative, Neutral)
* Generate sector-wise market insights

## 🧠 Features

* NLP preprocessing
* TF-IDF vectorization
* Sector classification model
* Sentiment analysis
* Data visualization dashboard

## 🛠️ Tech Stack

* Python
* NLTK / spaCy
* scikit-learn
* Matplotlib / Seaborn
* Streamlit

## 📁 Project Structure

* data/ → datasets
* src/ → core ML modules
* notebooks/ → experiments
* app/ → UI

## 🎯 Objective

To assist investors in understanding market trends using news sentiment.
